僕のゲーム事情

経歴

初めてプレイしたパソコンのゲームは Windows98 の 「パソコンやりたい放題 マウス編」。

初めてプレイしたオンラインゲームは「メイプルストーリー」。

初めてドハマりしたゲームは「サドンアタック」。

初めて課金したゲームも「サドンアタック」。

初めてプレイした有料ゲームは「Minecraft」。

初めてプレイしたSteamゲームは「Counter-Strike: Global Offensive」。

課金勢のあの頃

有料のゲームを買うという考えは全くなく、無料ゲームしかしていませんでした。無料オンラインFPSゲームのサドンアタックにドハマりしていました。
約３年ほどやっていましたが、お年玉を叩いて叩いて合計で５万円近くは使ったのかなと思います。死んでも後悔しきれません。
課金をし始めて１年半ほど経って、「これ、ゲーム買った方が安く済むんじゃないか。」ということに気づくことができ、有料ゲームの中で好きなゲームを探すようになりました。

Steam勢の今

Steamを始めてから２年経ちましたが、注ぎ込んだお金は１０万円近く。所有ゲーム数が６００個近くになりました。
全く嬉しくない話ではありますが、無料ゲームに課金をするよりも数倍とマシなお金の使い道かなと思ってますね。
また、コンシューマ機よりもパソコンの値段の方が高いですが、ゲーム本体の値段は確実にパソコンゲームの方が安いので（高頻度でセールがされますし、今では信用できる鍵屋も多いです。）、ゲームを買えば買うほどコンシューマ機よりも安いことが分かってきます。
今ではゲームコレクターという肩書を求めて、日々ゲームを積み上げています。
